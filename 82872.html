JavaScript

JavaScript

Bahasa pemrograman yang sangat populer untuk pengembangan web interaktif. Saya menggunakan JavaScript untuk membangun berbagai aplikasi web front-end dan back-end dengan Node.js, serta integrasi API dan automasi.

Python

Python

Bahasa yang sangat kuat dan serbaguna, ideal untuk pengembangan aplikasi, scripting, analisis data, dan machine learning. Saya memiliki pengalaman membangun aplikasi dengan Flask dan melakukan pemrosesan data dengan Pandas dan NumPy.

Go

Go (Golang)

Bahasa pemrograman yang dikembangkan Google untuk performa tinggi dan kemudahan pemrograman sistem. Saya menggunakan Go untuk membangun aplikasi server yang scalable dan microservices yang responsif.

Rust

Rust

Bahasa sistem modern yang fokus pada performa dan keamanan memori. Saya bereksperimen dengan Rust untuk mengembangkan kode-kode yang cepat dan aman, khususnya untuk aplikasi yang membutuhkan manajemen memori yang ketat.

HTML & CSS

HTML & CSS

Pondasi dari semua halaman web modern. Saya mahir dalam membuat desain responsif, antarmuka yang menarik dan mudah digunakan, dengan tata letak modern yang kompatibel di berbagai perangkat.
